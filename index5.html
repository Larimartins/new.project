Parágrafos h1 até h6

As tags <h1> até <h6> são usadas para definir titulos e subtitulos em uma página HTML. Indicando níveis de hierarquia, onde o <h1> é o principal e maior fonte e o de menor tamanho "menos importante".

Tag <p>

A tag <p> é usada para você colocar os paragráfos.

Marcador

A tag <mark> é usada para quando você quiser marcar alguma palavra ou frase do seu texto.
Exemplo: <p>Essa palavra irá ser marcada de <mark>amarelo</mark></p>

Tag Small

A tag <small> é usada para deixar o texto menor do que o tamanho normal. Geralmente usados para observações, notas de rodapé, etc.
Exemplo: <p> Termos e condições <small> aplicáveis </small></p>

Tag <i>

A tag <i> é usada para deixar a fonte em itálico.
Exemplo: <p>Essa frase tem uma palavra em <i>itálico</i></p>

Tag u

A tag <u> é usada para sublinhar o texto.
Exemplo: <p> Essa <u> palavra</u> está sublinhada</p>

Tag strong

A tag <strong> é usada para dar ênfase em alguma parte importante do texto.
Exemplo: <p>Essa palavra irá ser <strong>destacada</strong></p>

Listas

Para criar uma lista existem duas maneiras:
Lista ordenada <ol> (numerada)
Exemplo:

  1. Item 1
  2. Item 2

Lista não ordenada <ul> (com marcadores)
Exemplo:

  • Item 1
  • Item 2

A tag <li> serve para definir o item da lista


Exemplo: <ul> <li>Primeiro item</li> </ul>

Tag a

A tag <a> serve para criar um hiperlink para outra página ou recurso.
Exemplo: <a href="exemploaqui.com">Visitar site</a>

Tag hr

A tag <hr> serve para inserir uma linha horizontal para separar o conteúdo

Tag sub

A tag <sub> coloca o texto em subscrito (abaixo da linha base).Usado para fórmulas químicas, etc.
Exemplo:

H <sub>2</sub>0

H 20

Tag sup

A tag <sup> coloca o texto em sobrescrito (acima da linha base). Usado para expoentes, notas de rodapé.
Exemplo:

x <sup>2</sup>

x2

Tag blockquote

A tag <blockquote> Destaca um bloco de citação de outra fonte.
Exemplo:
Lorem ipsum dolor sit amet, consectetur adipisicing elit. Aperiam quisquam aut perferendis quo corrupti architecto quaerat explicabo voluptatibus placeat sit harum velit excepturi voluptate quam aliquid corporis amet, fuga maiores

"Esta é uma citação longa."
